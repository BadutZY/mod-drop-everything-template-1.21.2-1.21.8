**You can add up to 5 mobs in the settings, and you can also add 7 items to each mob.**

### Add Mob Setting

**You can add mob by typing the name of the mob.**

![addmob](https://cdn.modrinth.com/data/cached_images/c3cfca75ffca28e44315cafac7aca8444b723975.png)

### Add Item Setting

**This setting will appear when you add a mob. You can add item by typing the name of the item.**

![additem](https://cdn.modrinth.com/data/cached_images/3b7493eeab4169fee2b37e6b0ca4e1d51aedc155_0.webp)

### Drop Item Setting

**This setting will appear when you add an item. You can set the minimum and maximum number of items that will drop, and the chance of dropping.**

![itemdrop](https://cdn.modrinth.com/data/cached_images/d88ab6b8f61202c5de472fc17be54b783ce11081_0.webp)

![ps](https://cdn.modrinth.com/data/cached_images/22bc3b74a095f1860e0c826130a5b775016e49b3.png)

### Keybind 

**to open mod configuration, default key "M".**
